# Tacit — Second Brain

A personal knowledge canvas that captures what you read, remembers who you know, and lets you chat with everything you've saved.

Drop any URL — YouTube, TikTok, Instagram, articles, or any webpage — and Tacit extracts the content, summarises it, tags it, and places it on a visual canvas. Ask questions about your canvas in natural language. Tacit remembers people you mention and can search the web for anything not already in your knowledge base.

## Features

### Canvas

- **Drop any URL** — YouTube (with full transcripts), TikTok, Instagram, articles, and any webpage
- **Visual canvas** — pan, zoom, drag cards freely to organise your knowledge
- **Auto-generated edges** — semantic connections drawn automatically between related nodes based on content similarity
- **Manual linking** — ask Tacit in chat to link or unlink any two nodes
- **Category sidebar** — filter the canvas by topic; click any category to dim everything else
- **Auto-arrange** — group and cluster cards by category with one click
- **Duplicate detection** — normalises URLs and blocks re-adding the same content; highlights existing card and scrolls to it
- **Node detail panel** — click any card to see full transcript/content, tags, and connected nodes

### Content Extraction

- **YouTube** — official transcript API with yt-dlp subtitle fallback; captures title, duration, uploader, view count, thumbnail
- **TikTok & Instagram** — downloads audio via yt-dlp, transcribes with faster-whisper (runs on CPU)
- **Web articles** — trafilatura for clean text extraction, BeautifulSoup fallback, Playwright for JavaScript-heavy pages (X/Twitter, SPAs)
- **AI processing** — each node gets a title, 2–3 sentence summary, category, purpose, tags, and key entities generated by Claude

### Chat

- **Ask anything** — queries your entire canvas using semantic vector search (ChromaDB)
- **Real-time canvas awareness** — Tacit sees every node immediately after it's added, including ones still processing
- **Web search** — when the answer isn't on canvas, Tacit searches the web via Claude Haiku (no extra API key needed)
- **Temporal awareness** — "what did I add recently?" returns nodes in chronological order, not just by relevance
- **Source citations** — every response shows which nodes, documents, or web results it drew from
- **Multiple chat modes** — general, coaching (frames responses as a thinking partner), and direct knowledge query
- **Conversation history** — full multi-session history, grouped by Today / Yesterday / This Week / Older; switch between sessions
- **Persistent sessions** — conversation restored on page reload via localStorage

### People Memory

- **Automatic detection** — Tacit spots names mentioned in chat without any special command
- **Rich profiles** — stores role, organisation, relationship type, context paragraph, action items, and timestamped notes
- **Always-on recall** — relevant person context injected into every chat response that mentions them
- **People view** — browse all remembered contacts, see their details, delete any entry

### Infrastructure

- **Semantic search** — ChromaDB with cosine similarity across three collections (nodes, documents, contexts)
- **Agentic tool loop** — up to 5 iterations of tool use per response (canvas search, edge creation, people recording, web search)
- **SQLite + WAL mode** — concurrent read/write, survives restarts, no database server required
- **Async processing** — URL ingestion returns immediately; AI analysis runs in the background
- **Single Docker container** — FastAPI serves both API and frontend; one `docker compose up` to run everything
- **Data persistence** — all data in `./data/` on your machine; named volume for the embedding model cache

## Quick Start

### Docker (recommended)

**Requirements:** [Docker Desktop](https://www.docker.com/products/docker-desktop/)

```bash
git clone https://github.com/nik-sobolev/tacit
cd tacit
cp .env.example .env
```

Edit `.env` — set your Anthropic API key and your details:
```
ANTHROPIC_API_KEY=your_key_here
USER_NAME=Your Name
USER_ROLE=Your Role
USER_ORGANIZATION=Your Organization
```

```bash
docker compose up
```

Open **http://localhost:8000** — that's it.

Data (database, uploads, vector index) is stored in `./data/` on your machine and persists across restarts.

---

### Manual install

**Requirements:** Python 3.11+, ffmpeg

```bash
git clone https://github.com/nik-sobolev/tacit
cd tacit
cp .env.example .env   # edit with your API key + details

cd backend
pip install -r requirements.txt
playwright install chromium --with-deps
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

Open **http://127.0.0.1:8000**

## Stack

| Layer | Tech |
|-------|------|
| Backend | FastAPI + SQLite + ChromaDB |
| Frontend | Vanilla HTML / CSS / JS |
| AI — conversation | Claude Sonnet 4.6 |

## API

The backend exposes a REST API at `http://localhost:8000/api`:

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/ingest` | Add a URL to the canvas |
| `GET` | `/graph` | Full canvas graph (nodes + edges) |
| `POST` | `/chat` | Send a chat message |
| `GET` | `/conversations` | List conversation history |
| `GET` | `/people` | List remembered people |
| `DELETE` | `/people/{id}` | Delete a person from memory |
| `GET` | `/categories` | Node categories with counts |
| `GET` | `/insights` | Canvas stats and insights |
| `PUT` | `/nodes/{id}` | Update node title or position |
| `DELETE` | `/nodes/{id}` | Delete a node and its edges |
| `POST` | `/nodes/{id}/link/{target}` | Create a manual edge |
